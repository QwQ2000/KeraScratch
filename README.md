# 自制垃圾版 PyTorch

此项目用于 [ECNU](https://www.ecnu.edu.cn/) 计算机科学与技术系优化理论及方法课大作业。

目标为实现一个简易的神经网络结构，并能运行一些简单的神经网络训练项目。
